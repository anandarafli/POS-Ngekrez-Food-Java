# Sistem POS Ngekrez Food

Aplikasi kasir berbasis Java NetBeans dan MySQL.

## Cara Menjalankan:
1. Pastikan XAMPP sudah berjalan (Apache & MySQL).
2. Import file `db_ngekrez_pos.sql` ke phpMyAdmin.
3. Buka proyek ini di NetBeans.
4. Sesuaikan konfigurasi database di class `koneksi.Koneksi` jika diperlukan.
5. Run file `utama.FormLogin`.
